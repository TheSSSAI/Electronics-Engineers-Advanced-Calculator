Description

A feedback component that suggests the nearest standard E-series resistor value when a user enters a non-standard value. It is interactive, allowing the user to accept the suggestion with a single click.

Design System Alignment

Uses design system tokens for colors (primary scale), typography (font size), spacing, and borders to ensure consistency with other feedback and form components.
