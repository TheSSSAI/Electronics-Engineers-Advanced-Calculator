Responsive Specifications

Desktop Layout

  • Full-width header with content constrained by a max-width container.
  • Logo on the far left.
  • Primary navigation links are visible.
  • Search bar and theme toggle are on the far right.

Tablet Adaptations

  • The layout collapses to the mobile view at a specific breakpoint (e.g., 996px as per Docusaurus standards) to ensure touch-friendliness and prevent content wrapping.

Mobile Adaptations

  • Primary navigation links are hidden.
  • A hamburger menu button is displayed to toggle a sidebar navigation.
  • The search bar may be collapsed into an icon that expands on click to save space.

Accessibility Features
