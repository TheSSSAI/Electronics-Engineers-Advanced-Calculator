Description

A responsive layout for documentation articles, featuring a main content area and an optional, sticky table of contents on the right. It integrates molecular and atomic components like Admonition Blocks, Code Blocks, and a Doc Paginator.

Design System Alignment

This organism orchestrates multiple smaller components, all of which use the foundational design system tokens for typography, color, and spacing to create a cohesive content experience.

Responsive Specifications

Mobile Adaptations

  • Layout collapses to a single column for main content.
  • The Table of Contents (ToC) is hidden entirely to maximize content visibility.
  • The Doc Paginator at the bottom stacks vertically.

Tablet Adaptations

  • The Table of Contents remains hidden.
  • The main content area has sufficient horizontal padding.

Desktop Enhancements

  • A two-column layout is used, with the main content on the left and a sticky Table of Contents on the right.
  • The ToC provides at-a-glance navigation for the article.

Accessibility Features

Keyboard Support

The Table of Contents links and the Doc Paginator links are focusable and activatable via keyboard.
