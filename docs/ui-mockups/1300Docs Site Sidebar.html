Main Content Area

This is where the main documentation article content would go. On desktop screens, this area sits to the right of the persistent sidebar. On mobile screens, the sidebar is a drawer that overlays this content.

Try resizing your browser window to see the responsive behavior. Below 768px, the sidebar will disappear, and a hamburger menu will appear in the top header. Clicking it (or its label, in this demo) will open the sidebar drawer.

Component Variations

desktop-persistent

On viewports wider than 768px, the sidebar is always visible on the left, pushing the main content to the side. It remains in place as the user scrolls the content.

mobile-drawer

On viewports narrower than 768px, the sidebar is hidden off-screen. It is revealed by interacting with a toggle button (hamburger menu) and slides in from the left, covering the main content with a backdrop.
