Molecular Component: E-series Suggestion

Provides feedback and an interactive suggestion for a related input.

States

This component has two primary states: visible and hidden. The visible state is interactive.

Visible State

Shown when a non-standard value is entered. The input should use aria-describedby to link to this component's ID for accessibility.



Hidden State

Shown when the input value is standard, or no suggestion is needed. The .eseries-suggestion--hidden class is applied.

Component Variations

Default

The standard appearance of the suggestion component.
