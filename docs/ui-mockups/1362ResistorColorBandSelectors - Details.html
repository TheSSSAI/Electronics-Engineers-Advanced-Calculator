Description

A group of ColorBandSelector components arranged to represent the color bands of a resistor. The number of selectors is configurable based on the resistor type (3, 4, 5, or 6 bands). It is the primary input mechanism for the resistor color-to-value converter.

Design System Alignment

This component composes the atomic 'ColorBandSelector' and arranges them using the design system's spacing tokens. It also utilizes semantic color tokens for error states.

Responsive Specifications

Mobile Adaptations

  • The flex container direction changes to 'column', stacking the band selectors vertically.
  • Each selector stretches to the full width of the container for easier tapping.

Tablet Adaptations

  • Maintains the horizontal layout of selectors as space permits.

Desktop Enhancements

  • Horizontal layout provides a clear visual representation of a physical resistor's bands.

Accessibility Features

Keyboard Support

Users can Tab between each ColorBandSelector. Each selector is then operable via keyboard as defined in its atomic implementation.

Screen Reader Support

Each selector is preceded by a descriptive label (e.g., 'Band 1: Digit'). The error state message is programmatically associated with the component group.

Focus Management

Focus order follows the visual order of the bands, left-to-right on desktop and top-to-bottom on mobile.

Aria Attributes

  • aria-labelledby to associate labels with selectors.
  • aria-describedby on the group container to link to the error message in the error state.

Color Contrast

All labels and text meet WCAG 2.1 AA contrast ratios.
