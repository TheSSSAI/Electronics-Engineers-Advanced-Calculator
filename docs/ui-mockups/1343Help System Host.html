Getting Started

Welcome to the help system. Here you'll find information about all the features of the application, from basic arithmetic to advanced electronics modes.

  • To perform a calculation, type an expression like 5 + 3 * 2 and press Enter.
  • Use the DEG/RAD/GRAD button to switch between angle modes for trigonometric functions.
  • Explore specialized modes like Ohm's Law for electronics calculations.

Custom Modes

You can create your own calculators using the Custom Mode wizard. This allows you to define inputs, outputs, and the formulas that connect them. For example, a formula for the area of a circle would be area = pi * radius^2.
