Description

An organism component that provides a two-way conversion tool for resistor color codes. It allows users to either select colors to find a resistor's value ('Color to Value') or enter a value to find the corresponding color bands ('Value to Color').

Design System Alignment

This component is a composition of several molecular and atomic components, including ResistorColorBandSelectors, NumericInputWithPrefix, and EseriesSuggestion. It uses the design system's tokens for layout, typography, and color to maintain a consistent user experience.

Tablet Adaptations

Desktop Enhancements
