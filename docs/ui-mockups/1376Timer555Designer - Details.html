Description

A specialized calculator for designing 555 timer circuits in either Astable or Monostable mode. It takes user inputs for desired parameters and calculates the required component values, showing warnings for impractical results.

Design System Alignment

This component is an organism that composes the NumericInputWithPrefix and WarningIconWithTooltip atomic components. It utilizes the design system's color palette for states (info, warning, error), typography for labels and values, and spacing tokens for layout.

Responsive Specifications

Mobile Adaptations

  • The layout is single-column by default.
  • On very narrow screens (< 480px), the tabs for Astable/Monostable stack vertically instead of horizontally.
  • Padding is reduced on smaller screens to maximize content area.

Tablet Adaptations

  • Maintains the default desktop layout.

Desktop Enhancements

  • Clear separation between input and output groups.
  • Utilizes hover states for interactive elements like tabs.

Accessibility Features

Keyboard Support

The tab switch is implemented as a radio group or `role='tablist'` for full keyboard navigation (Tab to enter, Arrow keys to switch). All form inputs are keyboard accessible.
