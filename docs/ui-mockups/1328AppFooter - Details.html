Description

A global footer component for the application that provides access to legal documents such as 'Terms of Service' and 'Privacy Policy', along with a copyright notice.

Design System Alignment

This component uses the design system's color palette for background and text, typography for font styles, and the spacing system for padding and layout.

Responsive Specifications

Mobile Adaptations

  • The footer container stacks its content vertically.
  • Navigation links are stacked vertically for easy tapping.

Tablet Adaptations

  • At 768px and above, the layout switches to a horizontal arrangement.
  • The copyright notice and navigation links are positioned on opposite sides.

Desktop Enhancements

  • The component maintains the horizontal tablet layout, constrained by a max-width for readability.

Accessibility Features

Keyboard Support

All links are standard anchor tags and are focusable and activatable via Tab and Enter keys.

Screen Reader Support

The navigation links are wrapped in a semantic `

` element with an `aria-label` for clear context.

Focus Management

A clear, high-contrast focus ring is applied to links using `:focus-visible`.

Aria Attributes

  • aria-label for the `
    ` element.

Color Contrast

Text colors (`--color-neutral-gray-700` on `--color-neutral-gray-50`) meet WCAG AA contrast requirements.
