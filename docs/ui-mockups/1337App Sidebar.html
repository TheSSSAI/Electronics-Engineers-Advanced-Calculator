Dashboard

This is the main content area of the application. The sidebar component is displayed alongside it.

Component Variations & States

This section describes the different variants and states of the App Sidebar component.

Variant Name: "Desktop (Persistent)"

Description: On screen widths greater than 1023px, the sidebar is permanently visible on the left. It has a fixed width and pushes the main content to the right. This is the default view on desktop devices.

To view: Ensure your browser window is wider than 1024px.

Variant Name: "Mobile (Drawer)"

Description: On screen widths of 1023px or less, the sidebar is hidden off-screen by default. It can be opened by clicking the "hamburger" menu icon in the header. When open, it slides in from the left over the main content, and a dark overlay appears behind it. It can be closed by clicking the overlay, pressing the 'Escape' key, or clicking the hamburger icon again.

To view: Resize your browser window to be narrower than 1024px. The hamburger icon will appear in the header. Click it to see the "open" and "closed" states of the drawer.
