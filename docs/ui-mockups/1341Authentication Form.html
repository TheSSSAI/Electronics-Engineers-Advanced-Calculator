Screen Variations

Variant Name: "Default Login Form"

The standard form for existing users to log in. All fields are ready for input.

Variant Name: "Default Register Form"

The standard form for new users to create an account. Includes password confirmation and terms of service agreement.

Variant Name: "Loading State"

Shows the form after submission, while waiting for a server response. The submit button is disabled and displays a spinner to indicate activity.

Variant Name: "Error State"

Demonstrates the form after a failed submission. A form-level alert provides a general error, and specific fields can be highlighted with inline error messages.
