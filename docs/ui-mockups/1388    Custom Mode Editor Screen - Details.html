Responsive Specifications

Desktop Layout

  • Wizard is centered on the screen with a maximum width (e.g., 800px).
  • Layouts within steps can use multiple columns where appropriate (e.g., formula editor next to variable list).
  • Clear visual hierarchy with ample white space.

Tablet Adaptations

  • Maintains the desktop layout but may reduce horizontal margins.
  • Ensures all interactive elements have sufficient tap targets.

Mobile Adaptations

  • Wizard takes up the full width of the screen.
  • All layouts are single-column, stacking vertically.
  • Navigation controls are fixed or easily accessible at the bottom of the viewport.
