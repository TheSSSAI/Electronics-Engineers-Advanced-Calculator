Component Type

Description

A layout component that groups a label, a form control (like TextInput or Checkbox), and helper/error text together. It manages spacing, state, and accessibility associations.

Design System Alignment

Utilizes design system tokens for spacing, typography, and semantic colors (error state). It is designed to contain atomic components like TextInput and Checkbox.

Responsive Specifications

Mobile Adaptations

  • The component uses a single-column layout by default, which is ideal for mobile.
  • Label stacks above the control on all screen sizes.

Tablet Adaptations

  • Maintains the stacked layout for clarity and consistency.

Desktop Enhancements

  • The stacked layout provides clear vertical rhythm on larger screens.

Accessibility Features

Keyboard Support

The contained form control (e.g., TextInput) is keyboard accessible.
