Description

The main header for the documentation site. It is responsive, containing the site logo/title, a search bar, and a hamburger menu toggle for mobile navigation.

Design System Alignment

This component uses the design system's tokens for colors (neutrals), spacing, shadows, and typography. It acts as a container for the 'Search Bar' molecular component.

Responsive Specifications

Mobile Adaptations

  • Displays a hamburger menu button to toggle the sidebar.
  • Collapses the full search bar into a single icon button that would trigger a search modal.

Tablet Adaptations

  • At the 768px breakpoint, the layout switches to the desktop view.

Desktop Enhancements

  • Hides the hamburger menu toggle.
  • Displays the site logo or title.
  • Shows the fully expanded search bar component.

Accessibility Features

Keyboard Support

All interactive elements (hamburger menu, search icon/input) are focusable with Tab and activatable with Enter/Space.

Screen Reader Support

The main container is a semantic `

`. The hamburger and search icon buttons have descriptive `aria-label` attributes.

Focus Management

Visible focus indicators (`box-shadow`) are applied to all interactive elements on `:focus-visible`.

Aria Attributes

  • aria-label='Toggle navigation menu' on the hamburger button.
  • aria-label='Open search' on the mobile search icon.

Color Contrast

All default text and background colors meet WCAG 2.1 AA contrast requirements.
