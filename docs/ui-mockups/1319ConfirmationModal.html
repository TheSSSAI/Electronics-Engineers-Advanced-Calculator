Confirmation Modal

A modal dialog used to confirm critical user actions. Below are the different variants and states as they would appear on a desktop screen.

Variants

Destructive Action Variant

Used to confirm an irreversible action, such as deleting an account or a custom mode. The primary confirmation button uses the 'danger' color.

Delete Custom Mode?

This action is permanent and cannot be undone. All data associated with this mode will be lost forever.

Informational Variant

Used for standard confirmations, such as navigating away from a form with unsaved changes. The primary confirmation button uses the 'primary' color.

Discard Changes?

You have unsaved changes. Are you sure you want to leave this page?

States

Loading State

Shown after a user confirms an action while waiting for the API response. Action buttons are hidden and a loading indicator is displayed.

Deleting...

Please wait while your custom mode is being deleted. Do not close this window.

Processing...
