Description

A form control for managing a dynamic list of resistor values. It allows users to add new values via a numeric input, and edit or remove existing values from the list. It's the primary data entry interface for the Resistor Combination calculator.

Design System Alignment

This component utilizes the design system's tokens for colors (primary, neutrals, error), spacing, typography, and borders. It also uses atomic components like TextInput, Button, and Icon, inheriting their styling and states.
