Responsive Specifications

Desktop Layout

  • Two-column layout within a centered card.
  • Left column for input and resistor list.
  • Right column for action buttons and results.

Tablet Adaptations

  • Maintains the two-column layout but with reduced horizontal margins.

Mobile Adaptations

  • Collapses to a single-column layout.
  • Input section appears first, followed by the resistor list, and finally the results section.
  • Full-width interactive elements for easier tapping.

Accessibility Features

Keyboard Navigation

Logical tab order from the input field, to the add button, through the list items and their controls, and finally to the calculation buttons.

Screen Reader Support

Semantic markup (form, label, ul, li, button). ARIA labels on icon buttons (e.g., 'Remove 1000 Ohm resistor'). Result areas use aria-live regions.

Focus Management

Visible focus rings on all interactive elements. Focus returns to a logical element after an action (e.g., after deleting an item).

Aria Attributes

  • aria-label
  • aria-describedby
  • role='status'
  • aria-live='polite'

Color Contrast

All text and UI elements meet WCAG 2.1 AA contrast ratios.
