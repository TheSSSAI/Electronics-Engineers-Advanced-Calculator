Creating Custom Calculation Modes

The custom calculation mode feature is a powerful tool that allows you to build personalized calculators for your specific, recurring needs. This guide will walk you through the process, from defining variables to writing formulas. For more information, see the full API documentation.

Step 1: Defining Variables

The first step is to define your inputs and outputs. Each variable needs a unique name that starts with a letter and contains only alphanumeric characters.

  • Inputs: These are the values you will provide each time you use the calculator.
  • Outputs: These are the values the calculator will compute for you.

Tip: Use clear and descriptive variable names like `input_voltage` instead of `v1` to make your formulas more readable.

Step 2: Writing Formulas

Once your variables are defined, you can write the formulas. The syntax is straightforward and supports standard arithmetic, along with a library of scientific functions.

Ohm's Law Example
// Assign the result to the 'current' output variable
current = voltage / resistance;

Warning: Formulas are executed in a secure sandbox. Functions that access the network or filesystem are not permitted and will result in an error.

Step 3: Sharing Your Mode

After creating and saving your mode, you can export it as a JSON file to share with colleagues or back it up. Check out how on our community page.
