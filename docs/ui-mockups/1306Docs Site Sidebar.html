Screen Variations

Variant Name: "Desktop Persistent"

On screens wider than 1024px, the sidebar is permanently visible on the left, and the main content area takes up the remaining space. This is the default view you see on a large screen.

Variant Name: "Mobile Drawer"

On screens narrower than 1024px, the sidebar is hidden by default. A hamburger menu icon in the header is used to toggle the sidebar, which slides in from the left as a drawer. The main mockup above is interactive; if your screen is narrow, click the hamburger icon to see this in action.
