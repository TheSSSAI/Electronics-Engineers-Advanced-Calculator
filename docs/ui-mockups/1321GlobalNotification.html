Global Notification Component

This component provides global feedback to users. It includes a persistent top banner for states like 'offline' and temporary toasts for actions like 'save success' or 'syncing'.

All notifications are implemented with ARIA live regions to be announced by screen readers.

Component Variations

Variant: Offline Banner

A persistent banner at the top of the viewport for application-wide states. It uses role="status" and aria-live="polite".

Connection lost. Core functions are available offline.

Variant: Toasts

Temporary notifications that appear at the bottom of the screen to confirm actions or provide status updates. They use role="status" and aria-live="assertive" to be announced immediately.
