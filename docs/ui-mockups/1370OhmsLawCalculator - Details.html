Description

A self-contained calculator for Ohm's Law and the Power Law. The user enters any two of the four values (Voltage, Current, Resistance, Power), and the component automatically calculates the remaining two.

Design System Alignment

This organism component is composed of the 'NumericInputWithPrefix' and 'WarningIconWithTooltip' atomic/molecular components. It utilizes the design system's color palette for states (error, warning, read-only), spacing tokens for layout, and typography for labels and messages.

Responsive Specifications

Mobile Adaptations

  • On screens narrower than 768px, the 2x2 grid reflows into a single column.
  • Padding around the component is reduced to maximize content area.

Tablet Adaptations

  • Maintains the 2x2 desktop grid layout.

Desktop Enhancements

  • A 2x2 grid layout is used for optimal use of space and clear visual grouping of the four variables.

Accessibility Features

Keyboard Support

All input fields are focusable and editable using the keyboard. The tab order is logical, following the visual layout.

Focus Management

A clear, high-contrast focus ring is applied to input fields on focus, following the design system's `:focus-visible` standard.

Aria Attributes

  • `aria-describedby` is used to connect inputs to error messages and the main status message.
  • `aria-invalid='true'` is applied to inputs in an error state.

All text, labels, and UI feedback elements meet WCAG 2.1 AA contrast requirements.
