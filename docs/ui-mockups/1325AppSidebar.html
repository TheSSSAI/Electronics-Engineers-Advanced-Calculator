Main Content Area

This area represents the main view of the application, such as the calculator itself. On desktop, the sidebar is permanently visible. On mobile (screens less than 768px wide), the sidebar is hidden and can be opened by clicking the hamburger menu icon in the top left.

Component Variations

Variant Name : "desktop_persistent"

The sidebar is always visible on the left side of the screen on viewports wider than 768px. "

Variant Name : "mobile_drawer"

On viewports narrower than 768px, the sidebar is hidden by default and slides in from the left as a modal drawer, covering the main content with an overlay. "
